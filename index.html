Setup Guide

Get started in minutes

1

Create your paper repository

Start with our template that includes the workflow, or create your own private GitHub repository for your research paper.

Use Template
2

Add the workflow file

If you used the template, skip this step; this workflow is already included in the template. Otherwise, create .github/workflows/redpen-review.yml in your repo with this content:

redpen-review.yml
3

Add your OpenAI API key

In your repository, go to Settings → Secrets and variables → Actions and add a new secret named OPENAI_API_KEY.

4

Install the GitHub App

Add RedPen to your repository to enable comment triggers.

Install on GitHub
5

Request a review

Comment on any commit to trigger an AI review of your paper:

@RedPenApp review
